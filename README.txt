HOW TO RUN:

1) First start the PI server, which keep listening to request from the android application.
2) Now, Run the android application, launcher activity will ask you for PI ip address and port number, enter the details and click on connect button.
3) Once connection is established we can see temperature, ambient and blind position reading from the PI server and these values will get updated if there is significant change in temperature.
4) Now, Click on view rules for getting all fuzzy rules, we can add, delete or modify the rules by clicking respective buttons.
5) After updating the rules go back to the main activity to see the changes, the blind position values will change according to the new rules.
